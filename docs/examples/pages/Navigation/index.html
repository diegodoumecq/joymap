Gamepad Navigation Demo

Controls

  • D-pad - Navigate focus spatially / Move text cursor
  • L1 / R1 - Previous / Next (Tab-like)
  • L2 / R2 - Focus before/after scrollable container
  • X - Click / Editor Mode Toggle / Hold X for voice-to-text
  • Square - Cancel / Delete character / Uncheck checkbox
  • Right Stick - Scroll the focused container (vertical & horizontal)

Editor Mode

  • Date Input: D-pad up/down = day, left/right = month
  • Color Picker: D-pad up/down = hue, left/right = luminance
  • Range Input: D-pad left/right = ±1, up/down = ±10
  • Number Input: D-pad up/down changes value (respects step/min/max)
